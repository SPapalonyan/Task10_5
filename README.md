# Анализ посещаемости сайта — декабрь 2024

Решение учебного задания по анализу пользовательского поведения на сайте с помощью Python.

## Описание

Функция `website_analyse` принимает путь к CSV-файлу с данными о визитах и выводит четыре статистических показателя:

1. **Среднее количество просмотренных страниц** за все визиты.
2. **Стандартное отклонение** времени, проведённого на сайте.
3. **Коэффициент вариации** с интерпретацией (Высокая / Умеренная / Слабая вариабельность).
4. **t-тест** для проверки значимости разницы в поведении пользователей с мобильных устройств и десктопов.

## Стек

- Python 3.x
- pandas
- scipy

## Запуск

```bash
pip install pandas scipy
python website_analyse.py
```

## Входные данные

CSV-файл `website_visits_december_2024.csv` со столбцами:

| Столбец | Описание |
|---|---|
| Дата посещения | Дата визита в формате DD-MM-YYYY |
| День недели | Сокращённый день недели (пн, вт, …) |
| Время посещения | Время в формате HH:MM |
| Тип устройства | Мобильное или Десктоп |
| Просмотренные страницы | Количество страниц за визит |
| Время на сайте (сек) | Длительность визита в секундах |

## Пример вывода

```
1) Среднее количество просмотренных страниц: 17.49
2) Стандартное отклонение: 1442.80
3) Высокая вариабельность: 0.6924
4) Есть статистически значимая разница в количестве страниц: 0.000137
```
